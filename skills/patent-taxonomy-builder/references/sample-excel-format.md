# Excel样本格式说明

本文档说明 patent-taxonomy-builder 导入已标注专利样本时，Excel文件的预期格式。

---

## 文件要求

- **格式**：`.xlsx`（Excel 2007+）
- **编码**：UTF-8（Excel默认即可）
- **大小**：建议不超过10MB（约5000条专利）
- **Sheet**：使用第一个Sheet

---

## 列定义

### 必填列

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 专利号 | string | 专利公开号 | CN116599234A |
| 标题 | string | 专利标题 | 一种磷酸铁锂正极材料及其制备方法 |
| 摘要 | string | 专利摘要 | 本发明公开了一种磷酸铁锂正极材料... |

| 列名 | 类型 | 说明 | 示例 |

| 备注 | string | 标注备注 | 边界案例，同时涉及正极和电解液 |

## 分类标签格式

分类标签使用 `/` 分隔层级，对应分类体系的层级结构：

```
一级/二级/三级
```

**示例：**
- `锂离子电池/正极材料/磷酸铁锂`
- `锂离子电池/负极材料/硅基`
- `固态电池/氧化物电解质`
- `钠离子电池`

**规则：**
- 至少包含一级标签
- 层级用 `/` 分隔，不需要空格
- 标签名称应与分类体系中的分支名称一致
- 如标签与分支名不完全匹配，AI会尝试模糊匹配

## 导入流程

1. **文件验证**：检查文件格式、列名是否正确
2. **数据清洗**：去除空行、去重（按专利号）
3. **标签匹配**：将分类标签匹配到分类体系分支
4. **匹配报告**：生成匹配结果统计

### 匹配逻辑

```
对每条样本:
  1. 解析分类标签，拆分为层级路径
  2. 在分类体系中查找匹配的分支
  3. 精确匹配：标签路径与分支路径完全一致
  4. 模糊匹配：标签名称与分支名称部分匹配
  5. 关键词匹配：用摘要中的关键词辅助匹配
```

### 匹配结果分类

| 结果 | 说明 | 后续处理 |
|------|------|---------|
| 精确匹配 | 标签路径与分支路径一致 | 自动归入 |
| 模糊匹配 | 标签名称与分支名称相似 | 需用户确认 |
| 无法匹配 | 无对应分支 | 需用户指定或新增分支 |

## 常见问题

### Q: 分类标签与分支名称不一致怎么办？
A: AI会尝试模糊匹配。建议在标注时尽量使用与分类体系一致的名称。

### Q: 一条专利属于多个分类怎么办？
A: 在分类标签列用分号分隔，如 `锂离子电池/正极材料/磷酸铁锂;锂离子电池/电解液`。AI会识别为主分类+副分类。

### Q: 没有分类标签的样本可以导入吗？
A: 可以，但分类标签列为空时，AI会尝试根据摘要自动匹配，匹配结果需用户确认。

### Q: Excel列名不完全匹配怎么办？
A: AI会尝试识别常见变体（如"专利号"="公开号"="patent_id"），无法识别时会提示用户确认列映射。
